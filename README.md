# 🧹 大掃除マスター (Ōsōji Master)

Googleスプレッドシートと連携して、大掃除の計画・実行・記録を楽しく管理できるツールです。
「どこを掃除するか」を入力するだけで、AIが「具体的な内容」や「必要な道具」を自動で提案してくれます。

## ✨ できること
1.  **AIおまかせ入力**: 「キッチン」「換気扇」など場所を入れるだけで、掃除内容をAIが提案！
2.  **写真で記録**: 掃除後のピカピカになった状態を写真で保存できます（ブラウザ上での確認用）。
3.  **シート連携**: ボタン一つでスプレッドシートにリストを保存・読み込みができます。
4.  **スマホ対応**: 大掃除中はスマホでリストを見ながら作業できます。

---

## 🛠️ 導入手順（スプレッドシートで使う場合）

プログラミングの知識がなくても、以下の「コピー＆ペースト（貼り付け）」作業だけで設定できます。

### 1. スプレッドシートの準備
1.  新しいGoogleスプレッドシートを作成します（または既存のシートを開きます）。
2.  上部メニューの **「拡張機能」** > **「Apps Script」** をクリックします。
3.  新しいタブでプログラムの編集画面が開きます。

### 2. プログラムの貼り付け (`Code.gs`)
1.  編集画面にもともとある `Code.gs` というファイルをクリックします。
2.  中に入っているコード（`function myFunction...` など）をすべて消します。
3.  このアプリの **`Code.js`** ファイルの中身をすべてコピーして、そこに貼り付けます。
4.  ツールバーのフロッピーディスクのアイコン💾（保存）をクリックします。

### 3. 画面の貼り付け (`index.html`)
1.  「ファイル」の横にある **「＋」** ボタンをクリックし、**「HTML」** を選択します。
2.  ファイル名を **`index`** と入力してEnterキーを押します（`index.html`が作成されます）。
3.  もともと入っているコードをすべて消します。
4.  このアプリの完成版HTMLコード（※）をすべてコピーして、そこに貼り付けます。
5.  保存ボタン💾をクリックします。

> ※注意: 開発者から渡された「ビルド済み（1つにまとまった）HTMLコード」を使用してください。

### 4. アプリの起動
1.  スプレッドシートの画面に戻り、ブラウザの更新ボタン（F5キーなど）を押してページを**再読み込み**します。
2.  数秒待つと、メニューバーの右端に **「大掃除マスター」** という新しいメニューが表示されます。
3.  **「大掃除マスター」** > **「アプリを開く」** をクリックします。
4.  画面の右側にアプリが表示されます！

## 📱 使い方ガイド

### 基本操作
*   **行を追加**: 「＋行を追加」ボタンを押すと、新しい入力欄が増えます。
*   **場所を入力**: 「場所」の欄に「窓」「お風呂の床」などを入力します。
*   **AI提案（🪄杖のマーク）**: 場所を入力した後、紫色の杖マークをクリックすると、AIが掃除内容や道具を自動で入力してくれます。
*   **写真を添付**: カメラマークを押すと、掃除後の写真をアップロードできます。
*   **完了チェック**: 写真をアップロードするか、手動でチェックを入れると行がグレーになり「完了」扱いになります。

### データの保存
*   **シートに保存**: 緑色の「シートに保存」ボタンを押すと、現在のリストがスプレッドシートに書き込まれます。
*   **注意点**: 写真データは容量が大きいため、スプレッドシートには保存されません（アプリを開いている間のみ表示されます）。

## 🔑 AI機能について（管理者向け）
AI機能を使用するには「Gemini APIキー」の設定が必要です。
APIキーをお持ちでない場合や、AI提案ボタンが動かない場合は、システムの管理者にお問い合わせください。
